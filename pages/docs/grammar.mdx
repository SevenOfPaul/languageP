## 语法
+ 作为一门简单的玩具级解释语言
+ 鹏语言的语法基本对标js
### 注释
+ 既支持`//`注释也支持`/**/`注释[^ps1]
+ 例如
```ts
// 我是个单行注释
/**
 * 我是多行注释
 */
```
### 分支结构（条件语句[^ps2]）
+ 生活中总会一些如果 才 否则 的事情
+ 程序也是这样
```ts
if(true){
    print "真";
}else{
     print "假"; 
}
```
+ 当你也可以没有否则
```ts
if(true){
    print "真";
}
```
+ 当你也可以用中文
```ts
若(true){
    print "真";
}则{
     print "假"; 
}
```
+ 请原谅不支持else if (真不知道这个怎么起名了)

### 循环结构（循环语句）
+ 说起来计算机的出现就是为了解决重复的劳动
+ 但是目前为止还是我在教计算机写代码 
+ 下面进入重要内容`打印一百个"hello world!"`
+ 手写就很难了，是时候引入循环了。
```ts
//首先定义一个变量
let idx=1;
//while 相当于一个if
while(idx<=100){
    idx=idx+1;
    print "hello world";
}
```
+ 这样的话内部的`idx=idx+1`和`print "hello world!"`就会执行一百次了。
+ 当然毕竟鹏语言是对标c语言，那么for循环自然也是有的。
```ts
for(let i=0;i<100;i=i+1){
  print "hello world";
}
```
+ for循环相当于是while循环的语法糖[^ps3]。把变量声明 变量变化 变量比较写在了一行内。
