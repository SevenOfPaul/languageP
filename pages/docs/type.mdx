## 类型系统
+ 鹏语言是一门动态强类型语言。翻译成人话就是。
```ts
let str="1";
//这个str可以改成任意类型的语言
str=1; //√
str=nil; //√
//但是数据类型之间是没有类型转换的
//比如常见的计算操作符
2+"1";
//在js中会变成"21" 但是在鹏语言中会报错。 
```
## 都有哪些类型呢？
### Str 字符串
+ 字符串是存储字符（比如 "彭于晏"）的变量。
+ 字符串可以是引号中的任意文本。需要使用双引号包裹
<LinkButton title={"字符串"} href={"../wasm"}>
```ts
let str="彭于晏";
//当然也可以使用中文
诏 彭于晏="彭于晏";
```
</LinkButton>

### Num 数字
+ 鹏语言只有一种数字类型。数字可以带小数点[^ps1]，也可以不带。
<LinkButton title={"数字"} href={"../wasm"}>
```ts
let num=1;
print 1+num;
//当然也可以使用中文
诏 数=1;
打印 1+数;
```
</LinkButton>
### Bool 布尔值
+ 布尔只能有两个值：true 或 false。
+ 当然同样支持汉语 真，假
<LinkButton title={"布尔"} href={"../wasm"}>
```ts
let truthy=true;
let falsy=false;
//使用中文
诏 真的=真;
诏 假的=假;
```
</LinkButton>
### Time 时间[^p32]
+ 对的，你没看错。鹏语言将time作为基本数据类型
+ 这主要是为了从语言原生层面支持时间的计算
<LinkButton title={"Time"} href={"../wasm"}>
```ts
let start=now();
//算法....
let end=now();
//我早就想这么干了  hhh
let cousmeTime=end-start;
```
</LinkButton>
+ 不过time类型是不支持声明的 只能通过原生函数生成
```ts
//获取当前时间
let start=now();//√
let start=2017-03-12 // ×
```
### Nil 空值
+ 单纯的空 声明却未赋值的变量和没有返回值的函数 返回的都是这个值。相当于其他语言的null
<LinkButton title={"Nil"} href={"../wasm"}>
```ts
let start;
print start;//nil
```
</LinkButton>
## Func 函数
+ 是的，没错，在鹏语言中函数是一种基本类型。毕竟这门语言算是承继了js的血脉。
+ 可以写成非常爽的<Link className="title" href={"/talk/funcy"}>函数式</Link>代码
```ts
//声明
fn hello(){
    print "hello world!";
}
//调用
hello();
```
+ of course,函数不知这些，复杂的我们放到函数章节里单说。
