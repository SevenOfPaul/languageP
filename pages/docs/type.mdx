## 类型系统
+ 鹏语言是一门动态强类型语言。翻译成人话就是。
```ts
let str="1";
//这个str可以改成任意类型的语言
str=1; //√
str=nil; //√
//但是数据类型之间是没有类型转换的
//比如常见的计算操作符
2+"1";
//在js中会变成"21" 但是在鹏语言中会报错。 
```
## 都有哪些类型呢？
### Str 字符串
+ 字符串是存储字符（比如 "彭于晏"）的变量。
+ 字符串可以是引号中的任意文本。需要使用双引号包裹
<LinkButton title={"字符串"} href={"../wasm"}>
```ts
let str="彭于晏";
//当然也可以使用中文
诏 彭于晏="彭于晏";
```
</LinkButton>

### Num 数字
+ 鹏语言只有一种数字类型。数字可以带小数点[^ps1]，也可以不带。
<LinkButton title={"数字"} href={"../wasm"}>
```ts
let num=1;
print 1+num;
//当然也可以使用中文
诏 数=1;
打印 1+数;
```
</LinkButton>
3. Bool 布尔值
+ 布尔只能有两个值：true 或 false。
+ 当然同样支持汉语 真，假
<LinkButton title={"布尔"} href={"../wasm"}>
```ts
let truthy=true;
let falsy=false;
//当然也可以使用中文
诏 真的=真;
诏 假的=假;
```
</LinkButton>
4. Nil 空值
5. Func 函数（是的，没错，在鹏语言中函数是一种基本类型。）
6. Time 时间[^p32]
[^ps1]: *Num支持整数和浮点数，以RUST中f32为标准。但是注意 鹏语言的数据计算都是以浮点数（小数）为基准的，所以 0.1+0.2=0.3000000004这种事在鹏语言中也会出现。*
[^p32]: *以RUST中chrono中DateTime为标准。*
