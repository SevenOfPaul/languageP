## 类型系统
+ 鹏语言是一门动态强类型语言。翻译成人话就是。
```ts
let str="1";
//这个str可以改成任意类型的语言
str=1; //√
str=nil; //√
//但是数据类型之间是没有类型转换的
//比如常见的计算操作符
2+"1";
//在js中会变成"21" 但是在鹏语言中会报错。 
```
