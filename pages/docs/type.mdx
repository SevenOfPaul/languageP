## 类型系统
+ 鹏语言是一门动态强类型语言。翻译成人话就是。
```ts
let str="1";
//这个str可以改成任意类型的语言
str=1; //√
str=nil; //√
//但是数据类型之间是没有类型转换的
//比如常见的计算操作符
2+"1";
//在js中会变成"21" 但是在鹏语言中会报错。 
```
## 都有哪些类型呢？
1. Str 字符串
2. Num 数字[^ps1]
3. Bool 布尔值
4. Nil 空值
5. Func 函数（是的，没错，在鹏语言中函数是一种基本类型。）
6. Time 时间[^p32]
[^ps1]: 以RUST中f32为标准。
[^p32]: 以RUST中chrono中DateTime为标准。
