## 运算符
### 算术运算符
| 运算符      | 描述     |    例子   |  运算结果|
| ----------- | ----------- |------|---------|
| + | 加    |2+5|7|
|- | 减|9-10|-1|
|* | 乘 |12*4|48|
|/ | 除|22/2|11|
#### 对字符串进行加法运算
+ 没人能拒绝"hello"+"world"这样的操作;
+ 虽然鹏语言是强类型的语言，但是在字符串+数字上还是有隐式类型转换的。
+ `"2"+1="21"` 这里的1就被转化为了`"1"`之后与`"2"`相加;
#### 对时间进行运算
+ 鹏语言内部支持了对于时间的运算
+ 例如
```ts
let start=now();
//算法ing
let end=now();
print start-end;
print start+end;
print start>=end;
```

### 比较运算符
| 运算符      | 描述     |    例子   |  运算结果|
| ----------- | ----------- |------|---------|
| ==| 是否相等    |2==2|true|
| !=| 是否不等    |2!=2|false|
| >| 前者是否大于后者    |2>2|false|
| >=| 前者是否大于等于后者    |2>2|true|
| `<`| 前者是否小于后者    |`2<2`|false|
| `<=`| 前者是否小于等于后者    |`2<=2`|true|
### 三元表达式
+ 对的，鹏语言支持这么高级的特性。（写的时候废了老鼻子劲）
+ 三元表达式是<Link href="./grammar#分支结构条件语句" className="title"> ifElse </Link>的改写,?前是条件，后是第一个选项：后是第二个选项
+ 也就是说如果条件是真的，返回值为第一个选项，否则返回第二个选项。
```ts
let str=true?"对":"错";
print str ;//对
```
